JSP Intro Quiz

  1. What is the main value of Java Server Pages?

    Itâ€™s separate the display from processing, in other words, separate html from java.

  2. How are JSP pages related to servlets?

    JSP pages are servlets. When you run your application, the web container is going to compile JSP pages into servlet.

  3. How are JSP pages related to HTML?

    JSP can have html parts in the JSP page and it can be mixed with java code using expression languages, directives and tags.

  4. Give an example of a JSP scriptlet and show how it will look in the JSP servlet

    JSP scriplet: ------------- <% String greeting = "Hello World"; %> JSP servlet: ------------ String greeting = "Hello World"; greeting is a local variable;

  5. Give an example of a JSP declaration and show how it will look in the JSP servlet.

    - <%! int count = 0;%> These are declared as instance variables in the JSP Servlet.

  6. Give an example of a JSP comment and show how it will look in the JSP servlet.

    The example of comments in JSP looks like: <%-- comment --%>

  7. Give an example of a JSP expression and show how it will look in the JSP servlet

    <%= i*j+i %> The expression above will be compiled to the following in JSP servlet: out.print(i*j+i);

  8. Give an example of a JSP directive and show how it will look in the JSP servlet.

    <%@page import="mum.cs472.*" %> The directive above will be compiled to a proper import statement in the JSP servlet like: import mum.cs472.*

  9. Explain how an EL expression is evaluated.

    The servlet container will check on the page scope for that attribute and if find will print that attribute in the page html. If not found will check the request scope, session scope and application.

  10. Explain how servlet attributes are involved in EL expressions.

    If our servlet set the attributes in any scope (request, session etc), the attribute can be accessed inside the JSP by the name only without mentioning the scopes (scopes are implicit for the JSP) .

  11. Explain how servlets and JSPs use request dispatch to interact.

    Servlets handles the business logic and set attributes. After which, it forwards the request using a request dispatch to JSP.
